# 2026-04-27 Daily Log

## MindPal 截图问题已修复

### 问题
iPad Pro 13" M4 所有 Tab 截图 MD5 相同（都是首页），老爷要求每个 Tab 不同。

### 根因
iPad 有 floating tab bar 机制，每个 Tab 按钮会创建多个重叠元素（`_UIFloatingTabBarItemCell` 和 `_UIFloatingTabBarItemView`），导致 `app.buttons["Insights"]` 匹配到多个元素，XCUITest 拒绝执行 tap。

### 解决方案
1. 在 `MainTabBarController.swift` 给每个 TabBarItem 设置 `accessibilityIdentifier`：
   - `tab_journal`, `tab_insights`, `tab_growth`, `tab_settings`
2. XCUITest 用 `NSPredicate(format: "identifier == %@", identifier)` + `firstMatch` 精确匹配
