## 屏幕录制问题

### 问题
- macOS 15.7.3 屏幕录制权限限制
- 录制命令全部失败（SSH 和 VNC 环境）
- `screencapture -v -D 1` 需要屏幕录制授权
- ScreenCaptureKit 同样需要授权

### 尝试的方案
1. screencapture 命令 → 失败（权限）
2. ScreenCaptureKit Swift 程序 → 需要 macOS 13+，但在云 Mac 上有权限限制
3. Python 截图脚本 → 录到的是黑屏（App 未在 Simulator 中运行）

### 解决方案
- **需要用户在 VNC 上打开 Simulator 并运行 App**
- 然后使用 `screencapture -v -D 1 -T 60 ~/Desktop/demo.mov` 录屏
- 或使用 macOS 内置 QuickTime Player 录屏

### 关键提醒
- 录屏前必须先打开 HabitArcFlow App 在 Simulator 中运行
- 否则录到的只是黑屏
